Overview

The course registration systems allows you to enroll students, add instructors, add courses, register a student into a course, and drop a student from a course. From the home page, you can navigate to any form. As of now, these forms do not do anything.

HTML Pages

Home Page

The home page consists of seven links that each transport the user to their desired webpage. The order of the links are Enroll Student, Add Instructor, Register Course, Drop Course, User Manual, and Programmer Manual.

Enroll Student

The Enroll Student HTML page is a form that allows students to enroll in the institution. The form is made up of text files as well as a drop-down menu to select the studentâ€™s year. Each form is labeled to the left of the text field to help the user know what each field is meant to do. The user will start at the top of the page, fill out the appropriate forms, and click submit at the bottom. The user may also click the Home Button to return to the home page.

  • Form elements used:
    • Text
    • Button
    • Label
    • Drop Menu
  • Text and button element IDs
    • Student's_First_Name
    • Student's_Last_Name
    • Year
      • Freshman
      • Sophomore
      • Junior

    • Student's_Major
    • Student's_Email_Address
    • Buttons
      • SUBMIT
      • HOME BUTTON
  • Data validation rules
    • Student's First Name
      • Required Field
      • Must be a string. Spaces are allowed
      • Error message:
        • "Please enter your first name"
    • Student's Last Name
      • Required Field
      • Must be a string. Spaces are allowed
      • Error message:
        • "Please enter your last name"
    • Student's Year (i.e., Freshman, Sophomore, Junior, Senior)
      • No validation rules are necessary since this is a drop-down list.
    • Student's Major
      • Not required
      • Must be a string. Spaces are allowed
    • Student's Email Address
      • Required Field
      • Must follow the format abc123@marietta.edu. The first three characters must be letters. The next three characters must be numbers. The email address must end with @marietta.edu.
      • Error message:
        • "Please enter your email address"

Add Instructor

The Add Instructor page is a web form that allows you to add an instructor by entering their first name, last name, department and email address. You must also choose a rank for the instructor from the dropdown menu.

  • First Name
    • Required Field - Error: Please enter a first name
    • Must not be a number - Error: First name must be a string
  • Last Name
    • Required Field - Error: Please enter a last name
    • Must not be a number - Error: Last name must be a string
  • Department
    • Must not be a number - Error: Department must be a string
  • Email Address
    • Email must be in the form sssddd@marietta.edu where s are letters and d are digits - Error: Email in incorrect form
The back button at the bottom of the page will return you to the home page.

Add Course

The add course HTML page is a form which allows users to add a course to their schedule. The form is made up of text files mainly as well as dropdown menus to select answers with multiple options such as day, time, and semester of each course. Each of these fields are held within a form which sits inside the body and a header is placed at the top to help the user know where they are. Start from the top of the form and fill in the appropriate information as you go down until you are ready to submit. Be sure to fill in all fields.

Register Course

The Register Course page allows you to add a course for a student. This is done by inputting a student's first and last name. You then need to select a semester and year from the dropdown menu. You then enter the course prefix, number, and section. The submit button will see if your inputs are valid.

  • First Name
    • Required Field - Error: Please enter a first name
    • Must not be a number - Error: First name must be a string
  • Last Name
    • Required Field - Error: Please enter a last name
    • Must not be a number - Error: Last name must be a string
  • Year
    • Required Field - Error: Please enter a year
    • Must consist of four digits - Error: Year must consist of four digits
    • Year cannot be before 2023 - Error: Year cannot be before 2023
  • Course Prefix
    • Required Field - Error: Please enter a course prefix
    • Must have either three of four digits- Error: Course prefix can only be 3 or 4 capital letters
  • Course Number
    • Required Field - Error: Please enter a course numbe
    • Must be a three digit number no larger than 499 - Error: Course number must be a three digit number less than 500
  • Course Section
    • Required Field - Error: Please enter a course section
    • Must consist of exactly two digits - Error: Course section must be a two digit number
The back button at the bottom of the page will return you to the home page.

Drop Course

This page consists of six text boxes for the user to fill in their first name, last name, semester, course prefix, course number, and course section for the class they wish to drop. None of the text boxes store data currently but that will change in the future. There is also a submit button to submit future information as well as a return home link attop of the page to transport the user back to the home page.
